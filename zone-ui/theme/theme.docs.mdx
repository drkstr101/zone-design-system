## React Theme Context

This is a simple [React Context](https://reactjs.org/docs/context.html) shared as a Bit component.
Use this component to apply a theme as a context to set on it's children.

### Component usage

```tsx
() => {
import React, { useContext } from 'react';
import { ThemeProvider } from './theme-provider';
import { ThemeContext } from './theme-context';

<ThemeProvider themeMode="light">
  <MockComponent />
</ThemeProvider>;
```

### Using props to customize the theme

Change the themeMode to see the palette change:

```tsx live
() => {
  const theme = useContext(ThemeContext);
  return (
    <ThemeProvider themeMode="dark">
      <Paper sx={{ padding: 4 }}>
        <Button>this should be {theme.themeMode}</Button>
      </Paper>
    </ThemeProvider>
  );
};
```
